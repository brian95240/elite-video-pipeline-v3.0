## Deployment Options

### Local Development

```bash
# Clone repository
git clone https://github.com/brian95240/elite-video-pipeline-v3.0.git
cd elite-video-pipeline-v3.0

# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run tests
python tests/test_pipeline_closed_loop.py
```

### Docker Deployment

```bash
# Build image
docker build -t elite-pipeline-v3:latest .

# Run container
docker run -d -p 6379:6379 elite-pipeline-v3:latest
```

### Kubernetes Deployment

```bash
# Apply manifests
kubectl apply -f k8s/

# Verify deployment
kubectl get pods -l app=elite-pipeline
```
